# java 关键字 static final

## final

1. 修饰类，该类不能被继承，该类的方法默认都是final
2. 修饰方法，该方法不能重写，但可以继承
3. 修饰属性，该属性值不能改变
4. 修饰函数参数，可读取该参数，但不能改变其值

## static

1. 修饰属性（静态变量），在内存中只有一份拷贝，类级别
2. 修饰方法（静态方法），方法必须实现
3. 代码块
