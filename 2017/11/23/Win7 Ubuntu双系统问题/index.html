win7 ubuntu 双系统问题

安装Ubuntu时空闲的空间不可用

win7 分区状态 (系统 + C + D + E ) 四个全是主分区，导致最后压缩卷生成的空闲磁盘不能被Ubuntu识别

解决方案：

将最后一个磁盘 删除卷，在创建卷，自动变成了扩展分区，（内部变为逻辑分区）， 然后在压缩卷
（之前的压缩卷颜色是黑色的，更改完后变成绿色）
